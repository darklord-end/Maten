## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/darklord-end/Maten](https://github.com/darklord-end/Maten)
   cd Maten```

2. Установите зависимости:
    ```bash
    pip install -r requirements.txt```

3. Создайте файл `config.ini` и заполните его данными:
    ```ini
    [pyrogram]
    api_id = 123456789
    api_hash = abcdefghijklmnopqrstuvwxyz
    ```

4. Запустите бота:
    ```bash
    python3 main.py
    ```
